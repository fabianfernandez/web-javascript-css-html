Ejemplo de pagina hecha con HTML-Javascript-CSS

El objetivo de esta página web es poner en práctica las tecnologías de HTML5, CSS y Javascript. Esta web esta creada sin ningún Framework de diseño para poner en practica las bases asociadas al desarrollo web. Además, en esta página se puede encontrar un compilado de información, asociada a las tecnologías antes mencionadas para releer en caso de dudas o como forma de repaso.

HTML

HyperText Markup Language (Lenguaje de etiquetas de hipertexto), es un lenguaje de marcado para la elaboración de paginas web. Este lenguaje permite la creación de la estructura de una pagina web a través del uso de etiquetas.

CSS

Cascading Style Sheets (Hojas de estilo en cascada), es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Permite agregar estilos a las diferentes etiquetas del codigo HTML.

Javascript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.
