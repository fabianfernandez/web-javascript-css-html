Explicación general

El lenguaje de etiquetado HTML esta compuesto por una serie de nodos llamados etiquetas o elementos los cuales permiten decirle al navegador como mostrar el contenido de una pagina web.

Básicas

Las etiquetas mostradas a continuación son las más usadas:

  • La etiqueta <!DOCTYPE html> define que el documento es HTML5.
  • La etiqueta <html> es el elemento raiz de una pagina html.
  • La etiqueta <head> es el elemento que contiene los metadatos de la página.
  • La etiqueta <title> sirve para asignar el titulo a la pestaña de la web.
  • La etiqueta <body> define el cuerpo de la web donde se muestra el contenido.
  • La etiqueta <h1> define un titulo mostrado en el body.

Etiquetas

Las etiquetas tienen atributos los cuales dado cierto valor generan un comportamiento distinto. El atributo <style> sirve para agregar estilos a través de codigo CSS.

  • <h1> permite crear titulos y subtitulos, van desde h1-h6 dentro de una misma jerarquía. Ocupa todo el largo de su nodo padre. Los "headings" son usados por los motores de busqueda para indexar la página y que la web tenga mejor visibilidad, por lo que es importante utilizarlos de forma correcta y no solo para cambiar el aspecto de un titulo.
  • <p> permite crear parrafos.
  • <a> sirve para crear links.
  • <img> permite agregar imágenes. Nota: es mejor usar imagenes relativas a la página ya que no se tiene control de imagenes que provienen de otros dominios y es posible inflingir copyright
  • <br> agrega un salto de linea (Break)
  • <hr> etiqueta que rompe con una linea horizontal una página.
  • <pre> sirve para mostrar el contenido de la etiqueta con el formato preformateado, evitando que el navegador formatee el contenido.

Etiquetas de texto

  • <b> negrita
  • <strong> texto importante
  • <i> italica
  • <em> enfásis
  • <mark> texto marcado
  • <small> pequeña
  • <del> texto eliminado
  • <ins> texto insertado
  • <sub> texto subíndice
  • <sup> texto sobreescrito
